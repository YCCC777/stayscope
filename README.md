# StayScope

> 訂房前的地段分析助理 — Chrome 擴充功能

一鍵搜尋 Airbnb、Booking.com、Agoda、VRBO、Trip.com 房源周邊生活機能  
支援超市、藥局、餐廳、交通、景點等 20+ 類別，12 語系介面，零後端、零廣告。

---

## 功能亮點

| 功能 | 說明 |
|------|------|
| 🗺️ 座標提取 | 自動偵測各平台房源 GPS 座標 |
| 🔍 周邊搜尋 | 20+ POI 類別，直接開啟 Google Maps |
| ⭐ 收藏比較 | 最多 5 筆房源並排比較 + 地段 Checklist |
| 💰 價格追蹤 | 記錄每次開啟的價格，顯示漲跌趨勢 |
| 🎒 旅遊類型 | 背包客 / 親子 / 情侶 / 商務 / 銀髮 快選 |
| 🌍 12 語系 | 自動偵測瀏覽器語言切換 UI |

## 支援平台

| 平台 | 域名 |

## 安裝方式（開發測試）

1. `git clone https://github.com/YCCC777/stayscope.git`
2. Chrome 開啟 `chrome://extensions/`
3. 啟用右上角「開發人員模式」
4. 點擊「載入未封裝項目」，選取本資料夾

## 隱私政策

所有收藏儲存於使用者自己的 Chrome storage，不傳送至任何伺服器。無 Cookie、無追蹤、無廣告。

→ [完整隱私政策](https://yccc777.github.io/stayscope/privacy.html)
